# Version 5.1.0 (92)

## Add

- Timed 支援設定自動熱重載，大量邏輯最佳化
- 一個簡單的儲存綁定功能（後續擴充）：

設定目錄：**/data/adb/wipe_cache/BindConfigs**

在此目錄下建立一個自訂名稱的 **.conf** 檔案，填入自訂掛載規則：

```
<綁定目錄> <目標目錄>
```

註：請勿填寫**絕對路徑**，而是**相對內部儲存路徑**，否則會導致掛載失敗，暫不支援包含空格的路徑

當往 **<綁定目錄>** 內寫入/建立目錄/檔案時，將會被重新導向至 **<目標目錄>**

此功能暫時無法在使用 **sdcardfs** 的裝置上執行，但您無需擔心，**Android 12+** 基本上已無裝置使用 **sdcardfs**

## Update

- App UI 最佳化，升級目標 TARGET 至 Android 17，不再相容低於 Android 6.0 的裝置，修復 Pad 執行時彈出視窗異常拉伸
- 內部儲存清理、檔案分類使用 Android 標準儲存掛載路徑，測試修復 Android 12+ 幽靈檔案/資料夾問題
- 系統套件管理呼叫改回使用 PM 而非 CMD 以相容更多裝置
- 阻止快取白名單預設不再顯示系統 APP
- 刪除立即生效設定按鈕（已無用
- F2FS GC 改為背景執行+即時通知方式
