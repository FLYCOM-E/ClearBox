# Add
- App 返回桌面支援預測性返回手勢
- 檔案歸類/清理支援大小範圍指定：

配置目錄： **`/data/adb/wipe-cache/FileConfigs`**

在預設配置/自訂配置檔案內填寫 **`@MAX=<size/B/K/M/G>`**、**`MIN=<size/B/K/M/G>`** 可以指定目標檔案大小區間，覆蓋解析，後面的會覆蓋前面的設定，因此您可以利用此特性寫出您想要的規則

範例：

```
@MAX=1/G @MIN=10/M
zip 7z

@max=-1
iso img

@MIN=10/KB exe
```

以上幾種寫法均可，此宣告指明僅歸類最大 **1 G**，最小 **10 MB** 的 **zip**、**7z**，但 **iso**、**img** 則不限制最大大小

另：**-1** 表示無限制。**MIN** 不得大於 **MAX**，否則 **MIN** 自動失效。**MAX**、**MIN** 及大小單位是 **不區分大小寫** 的（您可能已經看出來了:⁠-⁠)），**暫不支援浮點數（小數）**
