# Version 5.1.0 (92)

## Add

- Timed 支持配置自动热重载，大量逻辑优化
- 一个简单的储存绑定功能（后续拓展）：

配置目录：**`/data/adb/wipe_cache/BindConfigs`**

在此目录下创建一个自定义名称 **.conf** 文件，填入自定义挂载规则：

```
<绑定目录> <目标目录>
```

注：请不要填写 **绝对路径**，而是 **相对内部储存路径**，否则会导致挂载失败，暂未支持带空格路径

当往 **<绑定目录>** 内写入/创建目录/文件时将被重定向至 **<目标目录>**

此功能暂无法在使用 **sdcardfs** 设备上运行，但您无需担心，**Android 12+** 基本已无设备使用 **sdcardfs**

## Update

- App UI 优化，升级目标 TARGET 至 Android 17，不再兼容低于 Android 6.0 设备，修复 Pad 运行时弹窗异常拉伸
- 内部储存清理、文件归类使用 Android 标准储存挂载路径，测试修复 Android 12+ 幽灵文件/夹问题
- 系统包管理调用改回使用 PM 而非 CMD 以兼容更多设备
- 阻止缓存白名单默认不再显示系统 APP
- 删除立即生效配置按钮（已无用
- F2FS GC 改为后台运行+实时通知方式
