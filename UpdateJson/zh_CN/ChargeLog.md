# Add
- App 返回桌面支持预测性返回手势
- 文件归类/清理支持大小范围指定：

配置目录： **`/data/adb/wipe-cache/FileConfigs`**

在默认配置/自定义配置文件内填写 **`@MAX=<size/B/K/M/G>`**、**`MIN=<size/B/K/M/G>`** 可以指定目标文件大小区间，覆盖解析，后面的会覆盖前面的设定，因此您可以利用此特性写出您想要的规则

栗：

```
@MAX=1/G @MIN=10/M
zip 7z

@max=-1
iso img

@MIN=10/KB exe
```

以上几种写法均可，此声明指明仅归类最大 **1 G**，最小 **10 MB** 的 **zip**、**7z**，但 **iso**、**img** 则不限制最大大小

另：**-1** 表示无限制。**MIN** 不得大于 **MAX**，否则 **MIN** 自动失效。**MAX**、**MIN** 及大小单位是 **非大小写敏感** 的（您可能已经看出来了:⁠-⁠)），**暂不支持浮点（小数）**
